```
docs/
├── README.md                    # 이 파일
└── 운세-서비스-기획서.md        # 운세 서비스 PRD
```

## 🎯 문서 작성 가이드

### 새 기획서 추가 시
1. `docs/` 폴더에 markdown 파일 생성
2. 파일명: `{프로젝트명}-기획서.md`
3. 이 README에 링크 추가
4. Git 커밋

### 기획서 버전 관리
- 주요 변경 시 버전 번호 업데이트
- 변경 이력을 문서 하단에 기록
- Git으로 이력 관리

## 📝 작성 규칙

- 모든 문서는 한글로 작성
- Markdown 형식 사용
- 명확하고 구체적으로 작성
- 다이어그램은 mermaid 또는 ASCII art 사용
